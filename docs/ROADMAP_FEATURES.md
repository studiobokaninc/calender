# 機能追加 実装ロードマップ

以下の8機能を Phase 1〜3 に分けて実装します。

---

## Phase 1（先行実装・依存少）

| 機能 | 内容 | 状態 |
|------|------|------|
| **グローバル検索** | ヘッダーに検索バーを設置。プロジェクト・タスク・イベントを横断検索し、結果から該当ページへ遷移 | 実装予定 |
| **ドラッグ&ドロップで日時変更** | FullCalendar のイベントドラッグ／リサイズを有効化し、変更時に API で更新 | 実装予定 |
| **ダッシュボード強化** | 今日の予定・今週の締切・遅延タスクを一覧で表示し、クリックでカレンダー／タスクへ遷移 | 実装予定 |

## 技術メモ

- **グローバル検索**: 既存 API（/projects, /tasks, /calendar/events）を利用し、フロントでフィルタまたは検索用エンドポイント 1 本を追加する案あり。
- **D&D**: FullCalendar の `editable: true`, `eventDrop`, `eventResize` を有効化し、イベント種別に応じて PUT /calendar/events またはタスク/プロジェクトの日付更新を呼び出す。
- **Google Calendar**: バックエンドに OAuth2 フローと Google Calendar API 呼び出しを実装。タスク一覧で「Google に表示」チェックで 1 件ずつ個人カレンダーに追加/解除。環境変数 `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`, `GOOGLE_REDIRECT_URI`（省略時は `http://localhost:8000/api/google/callback`）, `FRONTEND_URL`（連携後のリダイレクト先）を設定。
