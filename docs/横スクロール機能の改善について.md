# 横スクロール機能の改善について

## 問題の概要

別のパソコンからネットワーク経由でこのアプリにアクセスした際、マウスホイールでの横スクロールが効かない問題が発生していました。

### 原因

1. **ブラウザのセキュリティポリシー**
   - ネットワーク経由（localhost以外）でアクセスした場合、Chromeなどのブラウザは`wheel`イベントを自動的に`passive: true`として扱うことがあります
   - `passive: true`の場合、`preventDefault()`が無効化されるため、カスタムスクロール処理が動作しません

2. **HTTP vs HTTPS の違い**
   - HTTPSでない接続では、パフォーマンス保護のためにブラウザが追加の制限を適用することがあります

3. **リモートアクセス時の制限**
   - localhost以外からのアクセスでは、ブラウザが異なる挙動をすることがあります

## 実装した解決策

### 1. 自動横スクロール（最新版）

**横スクロールバーが表示されている場合、マウスホイールの上下操作が自動的に横スクロールに変換されます。**

**使い方：**
- 横スクロールバーが表示されているエリアでマウスホイールを上下に回転させるだけで横スクロールします
- **Shiftキーは不要**で、直感的に横スクロールできます
- ローカル環境でもネットワーク経由でも動作します

### 2. ダッシュボードの定期チェック機能

ダッシュボードページのチャット欄内のテーブルでは、以下の機能により確実に横スクロールできます：
- **初回表示時**: 100ms後にイベントリスナーを設定
- **定期チェック**: 500msごとに新しいテーブルをチェックしてイベントリスナーを設定
- **ページ切り替え後**: 他のページから戻ってきた時も自動的に横スクロール機能が有効化されます

### 3. ブラウザネイティブの横スクロールサポート

CSS設定により、ブラウザの標準的な横スクロールバーも使用できます。

## 対応済みのページ

以下のページで横スクロール機能が改善されました：

1. **TasksPage（タスク一覧）**
   - DataGridの横スクロールに対応

2. **Dashboard（ダッシュボード）**
   - メッセージテーブルの横スクロールに対応

3. **GanttView（ガントチャート）**
   - ガントチャートの横スクロールに対応

## 使用方法

### タスクページでの横スクロール

```
1. マウスホイールを上下に回転（Shiftキー不要）
   → 横スクロールバーが表示されている場合、自動的に横スクロールします

2. 横スクロールバーを直接ドラッグ
   → ブラウザ標準の操作で横スクロールできます
```

**重要**: 横スクロールバーが表示されていない場合は、通常通り縦スクロールします。

### ガントチャートでの横スクロール

```
1. Shiftキーを押しながらマウスホイールを上下に回転
   → 横方向にスクロールします

2. 矢印キーでの操作
   - ←/→: 横スクロール
   - ↑/↓: 縦スクロール
```

### ダッシュボードのテーブルでの横スクロール

```
1. テーブル上でマウスホイールを上下に回転（Shiftキー不要）
   → 横スクロールバーが表示されている場合、自動的に横スクロールします

2. ページ切り替え後も自動的に有効化
   → 他のページから戻ってきても、500msごとのチェックで自動的に横スクロール機能が設定されます
```

**重要**: ダッシュボードでは定期的にテーブルをチェックしているため、新しく追加されたテーブルでも自動的に横スクロールが有効になります。

### 主な変更点

#### TasksPage（最新版）

```typescript
// ✅ 改善後: 横スクロールバーが出ている場合は自動的に横スクロール
const handleWheel = (e: WheelEvent) => {
  const scrollableElement = container.querySelector('.MuiDataGrid-virtualScroller');
  if (!scrollableElement) return;

  // 既に横スクロールしている場合は通常の動作
  if (e.deltaX !== 0) return;

  // スクロール可能な範囲をチェック
  const hasHorizontalScroll = scrollableElement.scrollWidth > scrollableElement.clientWidth;
  
  // 横スクロールバーが出ている場合は、マウスホイールで横スクロール
  if (hasHorizontalScroll && e.deltaY !== 0) {
    e.preventDefault();
    scrollableElement.scrollLeft += e.deltaY;
  }
};
```

#### Dashboard（最新版）

```typescript
// ✅ 改善後: 定期チェックで確実にイベントリスナーを設定
useEffect(() => {
  const setupWheelHandlers = () => {
    const containers = Array.from(document.querySelectorAll('.message-content .table-scroll'))
    
    containers.forEach((el) => {
      // 既にイベントリスナーが設定されている場合はスキップ
      if ((el as any)._wheelHandler) return
      
      const onWheel = (e: WheelEvent) => {
        // 横スクロールバーが出ている場合は、マウスホイールで横スクロール
        if (el.scrollWidth > el.clientWidth && e.deltaY !== 0) {
          e.preventDefault()
          el.scrollLeft += e.deltaY
        }
      }
      
      el.addEventListener('wheel', onWheel, { passive: false })
      ;(el as any)._wheelHandler = onWheel
    })
  }

  // 初回設定
  setTimeout(setupWheelHandlers, 100)
  
  // 定期的にチェック（500msごと）
  const intervalId = setInterval(setupWheelHandlers, 500)
  
  return () => clearInterval(intervalId)
}, [messages, stateRestored])
```

#### イベントリスナーの設定

```typescript
// より互換性の高いイベントリスナーの設定
try {
  container.addEventListener('wheel', handleWheel, { passive: false });
} catch (e) {
  // 古いブラウザの場合はフォールバック
  container.addEventListener('wheel', handleWheel);
}
```

## トラブルシューティング

### 横スクロールが効かない場合

1. **横スクロールバーが表示されているか確認してください**
   - 横スクロールバーが表示されている場合のみ、マウスホイールで横スクロールできます
   - 表示されていない場合は、ウィンドウ幅を狭くするか、列を増やしてみてください

2. **ダッシュボードでは少し待ってください**
   - ページ切り替え後、最大500ms待つと自動的に横スクロール機能が有効になります

3. **ブラウザを更新してください**
   - ページをリロード（F5キーまたはCtrl+R）してみてください

4. **横スクロールバーを直接使用してください**
   - マウスやタッチパッドで横スクロールバーを直接操作できます

5. **別のブラウザを試してください**
   - Chrome、Firefox、Edgeなど、複数のブラウザで動作を確認してください

### ローカルとリモートでの動作

- **localhost（ローカル）**: すべての機能が正常に動作します
- **ネットワーク経由（リモート）**: 横スクロールバーが表示されている場合、マウスホイールで横スクロールできます
