# カレンダーアプリ 起動手順

## 前提条件
- Python 3.10以上がインストール済み
- Node.jsがインストール済み
- プロジェクトディレクトリ: `E:\calender`

## バックエンド（FastAPI）の起動

### 1. 仮想環境の有効化
```sh
cd /d E:\calender\backend
venv\Scripts\activate
```

### 2. サーバーの起動
```sh
uvicorn app.main:app --reload --port 8001
```

### 3. 起動確認
- ターミナルに以下が表示されれば成功：
```
INFO:     Uvicorn running on http://127.0.0.1:8001 (Press CTRL+C to quit)
INFO:     Application startup complete.
```

### 4. 停止方法
- ターミナルで `Ctrl + C` を押す

---

## フロントエンド（React/Vite）の起動

### 1. ディレクトリ移動
```sh
cd /d E:\calender\frontend
```

### 2. サーバーの起動
```sh
npm run dev
```

### 3. 起動確認
- ターミナルに以下が表示されれば成功：
```
VITE v6.2.5  ready in XXX ms
➜  Local:   http://localhost:5175/
```

### 4. 停止方法
- ターミナルで `Ctrl + C` を押す

---

## アクセス方法

### フロントエンド
- ブラウザで `http://localhost:5175` にアクセス

### バックエンドAPI
- APIエンドポイント: `http://127.0.0.1:8001`
- プロキシ設定により、フロントエンドから `/api/*` のリクエストが自動的にバックエンドに転送される

---

## 再起動が必要な場合

### バックエンドの再起動
1. 現在のサーバーを停止（`Ctrl + C`）
2. 上記の「バックエンドの起動」手順を実行

### フロントエンドの再起動
1. 現在のサーバーを停止（`Ctrl + C`）
2. 上記の「フロントエンドの起動」手順を実行

---

## 注意事項

- バックエンドとフロントエンドは**両方とも起動**している必要があります
- ポート番号は固定されているため、他のアプリケーションと競合しないよう注意してください
- 初回起動時は依存関係のインストールが必要な場合があります
